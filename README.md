# Distributed_Music_App

Decentralized music player based on IPFS Textile framework - IPFS.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

Clone repository

```
https://github.com/danyardelean/Distributed_Music_App.git
```

### Installing


Unzip the file and open the project using Android Studio IDE.



## Deployment

Works only with Android 7.0 or higher.

## Built With

* [Textile Android](https://github.com/textileio/android-textile/) - The IPFS framework used to store music
* [Firebase](https://firebase.google.com) - User authentification and Firestore NoSQL database
